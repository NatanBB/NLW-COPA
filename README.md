# :heavy_check_mark: Run Project:

#### --Server :globe_with_meridians:
```bash
cd server
```

```bash
npm install
```

```bash
npm run dev
```

#### --Web :computer:
```bash
cd web
```

```bash
npm install
```

```bash
npm run dev
```

#### --Mobile :iphone:

```bash
cd mobile
```

```bash
npm install
```

```bash
npx expo start
```

### Important :heavy_exclamation_mark:
```bash
Change .env.example to .env and fill the CLIENT_ID variable with your Google client id
```
